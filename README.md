# Lumina Lexical Analyzer

## About the project
This project is a simple web-based lexical analyzer built with HTML, CSS, and JavaScript. It takes source code as input and breaks it into tokens such as keywords, identifiers, operators, delimiters, and literals.

## The problem it solves
It helps students and developers quickly inspect how source code is tokenized. Instead of manually scanning code for lexemes, the app shows a structured table of the recognized tokens and their types.

## Technologies used
- HTML for the app layout
- CSS for the dark/light themed interface
- JavaScript for the lexical analysis logic
- Font Awesome icons for the UI

## Contributions
I created the full interface, implemented the lexer logic, added support for keywords and operators, built the token display table, and included features such as file upload, code clearing, and theme toggling.

## How to run
1. Open the project folder in a browser.
2. Launch the index.html file directly, or use a simple local server if preferred.
3. Paste or upload source code, then click Analyze to view the tokens.

## Interface
The app features a clean split screen:
- a source code editor on the left
- a token results table on the right
It also includes buttons for uploading files, analyzing code, clearing the input, and switching between dark and light themes.
